## 📊 项目概览

### 1. AI智能问答助手 (ai-qa-assistant)
- **技术栈**: Python, LangChain, OpenAI API, Streamlit, ChromaDB
- **功能**: RAG智能问答、多轮对话、文档检索
- **亮点**: 向量数据库集成、对话历史管理

### 2. 个人智能笔记管理系统 (smart-note-manager)
- **技术栈**: Python, Flask, SQLite, OpenAI API, TF-IDF
- **功能**: AI摘要、语义搜索、自动标签
- **亮点**: 智能分类、内容检索优化

### 3. AI写作助手工具 (ai-writing-assistant)
- **技术栈**: Python, Streamlit, OpenAI API
- **功能**: 文章生成、润色、翻译、格式优化
- **亮点**: 6种写作工具、参数定制

### 4. 智能日程管理助手 (smart-scheduler)
- **技术栈**: Python, Flask, SQLite, OpenAI API
- **功能**: AI建议、冲突检测、优先级管理
- **亮点**: 时间优化算法、智能提醒

- Python 3.10+
- OpenAI API密钥
- Git

### 安装步骤
`ash

git clone https://github.com/Max0633/ai-intern-projects.git
