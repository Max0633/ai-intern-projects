# AI Agent 实习项目集合

## 项目概述

这是我为AI Agent相关实习准备的项目集合，展示我在人工智能、大语言模型应用、全栈开发等方面的技术能力。

## 项目列表

### 1. AI智能问答助手 (ai-qa-assistant)
- **技术栈**: Python, LangChain, OpenAI API, Streamlit
- **功能**: 基于大语言模型的智能问答系统，支持多轮对话和知识检索
- **亮点**: RAG技术、向量数据库、对话历史管理

### 2. 个人智能笔记管理系统 (smart-note-manager)
- **技术栈**: Python, FastAPI, SQLite, 向量数据库
- **功能**: 智能笔记分类、搜索、摘要生成
- **亮点**: 自动标签、语义搜索、AI摘要

### 3. AI写作助手工具 (ai-writing-assistant)
- **技术栈**: Python, Streamlit, OpenAI API, Markdown
- **功能**: 文章生成、润色、翻译、格式优化
- **亮点**: 多种写作模式、实时预览、模板系统

### 4. 智能日程管理助手 (smart-scheduler)
- **技术栈**: Python, Flask, Google Calendar API, AI优化
- **功能**: 智能日程安排、冲突检测、优先级管理
- **亮点**: AI建议、时间优化、提醒系统

## 技术栈总览

- **编程语言**: Python 3.10+
- **AI框架**: LangChain, OpenAI API
- **Web框架**: Streamlit, FastAPI, Flask
- **数据库**: SQLite, ChromaDB (向量数据库)
- **开发工具**: Git, Docker, VS Code

- Python 3.10+
- pip 包管理器
- OpenAI API密钥 (部分项目需要)

### 安装步骤
`ash

# 进入项目目录
cd ai-intern-projects

# 安装依赖
pip install -r requirements.txt

# 运行项目
streamlit run app.py
`
